# partial-resolutions


This is a repository for auxiliary files accompanying my thesis,
'On terminal partial resolutions of symplectic quotient singularities'.
(After submission and acceptance the thesis will be published on arxiv.org,
and the relevant links to it will be provided here.)

These files primarily consist of code for the Magma programming language,
that give the results detailed in my thesis.
(This is ongoing as I am currently in the process of making corrections to my thesis.)
